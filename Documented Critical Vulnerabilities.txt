Documented Critical Vulnerabilities
1. Node.js Multiple Vulnerabilities
Plugin ID: 190856

Severity: Critical (CVSS v3: 9.8)

Description: Multiple high-severity vulnerabilities exist in outdated Node.js versions (18.x < 18.19.1, 20.x < 20.11.1, 21.x < 21.6.2). These could allow remote code execution or denial-of-service.

Fix/Mitigation: Upgrade Node.js to version 18.19.1, 20.11.1, or 21.6.2 (or newer).

Reference: Tenable Plugin 190856
